# 写真配置ガイド（仙豆のちから 姫路大手前通り店）

## 必要な写真ファイル

以下のファイル名で `public/images/` に写真を配置してください。
写真が準備できたものから順に反映されます（なくてもエラーにはなりません）。

### 1. ヒーロー（ファーストビュー）— 必須度: ★★★
- **ファイル名**: `hero.jpg`
- **推奨サイズ**: 横1920px以上、アスペクト比16:9程度
- **内容**: 店内の全体的な雰囲気、またはリクライニングチェアの引き写真
- **使い方**: 画面全体の背景として暗いオーバーレイ越しに表示
- **ポイント**: 暗くしても雰囲気が伝わる写真が◎。明るすぎる写真は避ける

### 2. 施術イメージ（お悩みセクション）— 必須度: ★★
- **ファイル名**: `treatment.jpg`
- **推奨サイズ**: 横800px以上
- **内容**: 施術中のイメージ（頭をほぐしている手元、リラックスしている様子など）
- **使い方**: お悩みセクションの装飾として小さめに表示
- **ポイント**: 顔がはっきり映らない角度がベスト（プライバシー配慮 + 洗練感）

### 3. 店内・個室（特徴セクション）— 必須度: ★★
- **ファイル名**: `interior.jpg`
- **推奨サイズ**: 横800px以上
- **内容**: 個室の雰囲気、チェアの様子、照明の雰囲気
- **使い方**: 特徴セクションの横に添える程度の控えめな配置
- **ポイント**: 落ち着いた照明で撮れているものが◎

### 4. アクセスセクション用（オプション）— 必須度: ★
- **ファイル名**: `exterior.jpg`
- **推奨サイズ**: 横800px以上
- **内容**: 外観・入口の写真（場所が分かるもの）
- **使い方**: 地図の横に小さく配置

## 注意事項
- JPG or WebP推奨（PNGは重い）
- お客様の顔がはっきり映っている写真は避ける
- 暗い雰囲気の写真の方がサイトのトーンに合います
